dogmas_magic
============
A web-app for rendering spell diagrams.

Usage
=====
1. Clone the repo
2. Spin up a small web server using something like `python -m SimpleHTTPServer`
3. Browse to `http://localhost:8000/magic.html`

Some spells to try:
* (fire)
* (fire/water)
* ((fire)/[[water/air]/earth])
